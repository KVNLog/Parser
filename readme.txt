
# Инструкция по запуску скриптов

## 1. Установка Python
- Скачайте и установите Python с официального сайта: [https://www.python.org/downloads/](https://www.python.org/downloads/).
- Во время установки обязательно отметьте галочку **"Add Python to PATH"**.
- После установки проверьте, что Python установлен корректно.
  - **Windows**: Нажмите `Win + R`, введите `cmd` и нажмите Enter. Затем выполните команду:
    ```bash
    python --version
    ```
  - **Linux/macOS**: Откройте терминал (например, с помощью сочетания клавиш `Ctrl + Alt + T`) и выполните:
    ```bash
    python3 --version
    ```

## 2. Установка pip
- Обычно `pip` (менеджер пакетов Python) устанавливается вместе с Python. Проверьте его наличие:
  ```bash
  pip --version
  ```
- Если `pip` не установлен, следуйте [официальным инструкциям](https://pip.pypa.io/en/stable/installation/).

## 3. Создание виртуального окружения
- Перейдите в папку с проектом (например, `Parser`).
  - **Windows**: В командной строке выполните:
    ```bash
    cd путь_к_папке_со_скриптами
    ```
  - **Linux/macOS**: В терминале выполните:
    ```bash
    cd путь_к_папке_со_скриптами
    ```
- Создайте виртуальное окружение:
  ```bash
  python -m venv venv
  ```
- Активируйте виртуальное окружение:
  - **Windows**:
    ```bash
    venv\Scripts\activate
    ```
  - **Linux/macOS**:
    ```bash
    source venv/bin/activate
    ```

## 4. Установка зависимостей
- Убедитесь, что виртуальное окружение активировано (в командной строке или терминале должна появиться надпись `(venv)` перед курсором).
- Установите зависимости из файла `requirements.txt`:
  ```bash
  pip install -r requirements.txt
  ```

## 5. Работа с Excel
- Для работы скриптов необходим файл `Рабочий.xlsx`, убедитесь, что файл находится в папке проекта.
- Открыть файл в эксель.
- в файле должен быть только один лист, если есть другие листы их лучше удалить.
- в первый столбец вставить перечень артикулов для поиска. Важно!: Не должно быть названия столбца, иначе скрипт посчитает название как артикул.
- В результате должен получиться файл эксел, с одним листом, в первом столбце должен быть перечень артикулов.

## 6. Запуск скриптов
- Находясь в активированном виртуальном окружении, запустите необходимый скрипт, указав его название. Например:
  ```bash
  python ChipDipParser.py
  ```
- Для запуска других скриптов замените `ChipDipParser.py` на имя соответствующего файла.

## 7. Проверка и устранение ошибок
- Если возникают ошибки:
  - Убедитесь, что зависимости установлены (см. пункт 4).
  - Проверьте, что виртуальное окружение активировано.
  - Удостоверьтесь, что используемая версия Python соответствует требованиям проекта.
- Для подробной информации об ошибках проверьте сообщения в терминале.
- Обратитесь к администратору
